# Dog Sitting Web Application

## Overview
This project is a full-stack web application that allows users to manage dog sitting information, including dog and owner details. It was built as an 
introduction to developing a traditional web application with a connected database.

## Live Demo
Access the application here:
https://cis-linux2.temple.edu/tomcat10/fa25_3308_tuq44019/#/home

## Features
- Store and manage dog and owner information
- Database integration using SQL
- Full-stack functionality with front-end and back-end components

## Technologies Used
- Frontend: React, JavaScript, HTML, CSS
- Backend: Java
- Database: MySQL

## Purpose
This was my first full web application project, where I focused on understanding how front-end interfaces interact with back-end logic and databases in a 
real-world scenario.
